# 🟣 Entities Layer

## 역할
비즈니스 엔티티(도메인 모델)를 관리하는 레이어입니다.

## 포함 내용
- `user/` - 사용자 엔티티
- `exam/` - 시험 엔티티
- `question/` - 문제 엔티티
- `answer/` - 답안 엔티티
- `grade/` - 성적 엔티티

## 예시
```
entities/
├── user/
│   ├── model/     # 사용자 타입, 스키마
│   ├── api/       # 사용자 API
│   ├── ui/        # 사용자 카드 등
│   └── lib/       # 사용자 유틸리티
└── exam/
    └── ...
```

## 의존성 규칙
- shared에만 접근 가능
- 모든 상위 레이어에서 접근 가능
- 다른 entities 간 직접 참조 금지
